Apache Tomcat (abans sota el projecte Apache Jakarta) es un contenidor de servlets desenvolupat a l'Apache Software Foundation. Tomcat implementa les especificacions de servlet i de JavaServer Pages (JSP) de Sun Microsystems, proporcionant un entorn per al codi Java a executar en cooperació amb un servidor web. Aquest afegeix eines per a la configuració i el manteniment, pero també pot ser configurat editant els fitxers de configuració que normalment son en format XML. Tomcat inclou el seu propi servidor HTTP, per aixo se'l considera un servidor web independent.
